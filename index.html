Rules:

  • This is a dice rolling game. The numbers between 2 and 6 adds to the current points of each player.
  • The points will sum until the dice rolls 1, if that happens, the current number will be erased.
  • In order to save the points, the user has to press on the HOLD button. That will make the points to be stored on the main score.
  • The first player to reach 100 points wins.
